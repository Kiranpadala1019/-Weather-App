# 🌦️ Django Weather App

A real-time weather forecasting web application built using **Python Django**, **Bootstrap**, and **OpenWeather API**.

Users can search any city and get live weather information including temperature, humidity, wind speed, and weather conditions.

---

## 🚀 Features

- 🌍 Search weather by city name
- 🌡️ Real-time temperature data
- 💧 Humidity information
- 🌬️ Wind speed details
- ☁️ Weather condition display
- 🖼️ Dynamic weather icons
- 📱 Responsive Bootstrap UI
- ❌ Invalid city error handling
